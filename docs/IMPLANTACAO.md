# Implantação

1. Execute os arquivos da pasta `sql` em ordem numérica no SQL Editor do Supabase.
2. Configure no Supabase a URL final do site e os redirecionamentos `/painel` e `/redefinir-senha`.
3. Importe o repositório na Vercel como projeto estático.
4. Teste cadastro, confirmação de e-mail, login, logout, recuperação de senha, lançamentos e projeção de 12 meses.
5. Valide o acesso em smartphone e computador antes de liberar usuários reais.
